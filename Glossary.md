# Glossary - Graphics Programming Terms

[← Back to Main Index](README.md)

Quick reference for common graphics programming terms and acronyms.

---

## A

**ADPCM** - Adaptive Differential Pulse Code Modulation, audio compression format

**ALU** - Arithmetic Logic Unit, performs mathematical operations in shaders

**Anti-Aliasing** - Techniques to reduce jagged edges (MSAA, FXAA, TAA, SMAA)

**API** - Application Programming Interface

**ASTC** - Adaptive Scalable Texture Compression, mobile texture format

**Attenuation** - Reduction of light intensity over distance

---

## B

**Batching** - Combining multiple draw calls into one to reduce CPU overhead

**BC (Block Compression)** - DirectX texture compression formats (BC1-BC7)

**Billboard** - 2D sprite that always faces the camera

**Bloom** - Post-processing effect creating glow around bright areas

**Bottleneck** - Performance limitation in CPU, GPU, memory, or bandwidth

**Burst** - Unity's LLVM-based compiler for high-performance C# code

---

## C

**Cascaded Shadow Maps (CSM)** - Technique for high-quality shadows at varying distances

**Clipping** - Removing geometry outside the view frustum

**Compute Shader** - GPU program for general-purpose computation

**Culling** - Process of eliminating objects not visible to the camera

---

## D

**Deferred Rendering** - Rendering technique storing geometry data in G-buffer

**DLSS** - Deep Learning Super Sampling, NVIDIA's AI upscaling technology

**DOTS** - Data-Oriented Technology Stack, Unity's ECS implementation

**Draw Call** - Command to GPU to render geometry

**DXR** - DirectX Raytracing

**DXT** - S3 Texture Compression, legacy name for BC formats

---

## E

**ECS** - Entity Component System, data-oriented programming paradigm

**Early-Z** - Depth testing before fragment shader execution

---

## F

**FBX** - Filmbox, 3D asset interchange format

**Forward Rendering** - Rendering technique calculating lighting per-object

**FPS** - Frames Per Second, measure of rendering performance

**Fragment Shader** - GPU program processing pixels (pixel shader)

**Frustum** - View volume visible to camera

**FSR** - FidelityFX Super Resolution, AMD's upscaling technology

---

## G

**G-Buffer** - Geometry Buffer used in deferred rendering

**GI** - Global Illumination, indirect lighting simulation

**GNM/GNMX** - PlayStation low-level graphics APIs

**GPU** - Graphics Processing Unit

---

## H

**HDRP** - High Definition Render Pipeline, Unity's high-end rendering pipeline

**HDR** - High Dynamic Range, extended luminance range

**HLOD** - Hierarchical Level of Detail

**HLSL** - High-Level Shading Language, Microsoft's shader language

---

## I

**Impostor** - Low-cost billboard replacement for distant objects

**Instancing** - Drawing multiple copies of geometry in single draw call

---

## L

**LLGD** - Low-Level Graphics Debugger for consoles

**LOD** - Level of Detail, quality reduction for distant objects

**LUT** - Look-Up Table, used for color grading

---

## M

**Mipmap** - Pre-calculated texture sizes for different distances

**MRT** - Multi-Render Target, rendering to multiple textures simultaneously

**MSAA** - Multisample Anti-Aliasing

---

## N

**Nanite** - Unreal Engine 5's virtualized geometry system

**Normal Map** - Texture storing surface normal data for lighting detail

**NVN** - Nintendo Switch graphics API

---

## O

**Occlusion Culling** - Hiding objects blocked by other objects

**OIT** - Order-Independent Transparency

**Overdraw** - Pixels rendered multiple times in same frame

---

## P

**PBR** - Physically Based Rendering, realistic material system

**PCIe** - Peripheral Component Interconnect Express, CPU-GPU connection

**PIX** - Performance Investigator for Xbox, Microsoft's profiling tool

**PSSL** - PlayStation Shader Language

---

## R

**Rasterization** - Converting vector geometry to pixels

**Ray Tracing** - Rendering technique simulating light rays

**Razor** - PlayStation profiling tool

**RenderDoc** - Open-source graphics debugger

**ROP** - Render Output Unit, GPU component for final pixel output

---

## S

**SIMD** - Single Instruction, Multiple Data

**SIMT** - Single Instruction, Multiple Threads

**SM** - Streaming Multiprocessor, NVIDIA GPU compute unit

**SRP** - Scriptable Render Pipeline, Unity's customizable rendering

**SSAO** - Screen Space Ambient Occlusion

**SSR** - Screen Space Reflections

---

## T

**TAA** - Temporal Anti-Aliasing

**Tessellation** - Subdividing geometry for detail

**TMU** - Texture Mapping Unit, GPU texture sampling hardware

**TRC/TCR** - Technical Requirements Checklist, console certification requirements

---

## U

**URP** - Universal Render Pipeline, Unity's cross-platform rendering

**UV** - Texture coordinates (U=horizontal, V=vertical)

---

## V

**VFX** - Visual Effects

**VRAM** - Video Random Access Memory, GPU memory

**VRR** - Variable Refresh Rate (FreeSync, G-Sync)

**VSync** - Vertical Synchronization, frame pacing technique

---

## W

**Warp** - NVIDIA term for group of threads executing together

**Wavefront** - AMD term for group of threads executing together

---

## X

**XeSS** - Intel's AI-based upscaling technology

---

## Z

**Z-Buffer** - Depth buffer storing pixel depth values

**Z-Fighting** - Visual artifact from insufficient depth precision

**Z-Prepass** - Early depth pass for optimization

---
