# Chapter 2: GPU Hardware Architecture

[← Back to Main Index](../README.md)

Understanding GPU hardware architecture is crucial for writing efficient graphics code. This chapter explores how modern GPUs work internally and how console architectures differ.

---

## Subjects in This Chapter

### [2.1 GPU Memory Hierarchy](../subjects/02-01-GPU-Memory-Hierarchy.md)
The organization of GPU memory including VRAM, L1/L2 cache systems, shared memory, constant memory, texture cache, and register files.

### [2.2 Execution Models](../subjects/02-02-Execution-Models.md)
How GPUs execute shader code including SIMD/SIMT architecture, warp/wavefront execution, thread divergence, GPU occupancy, and streaming multiprocessors.

### [2.3 GPU Components](../subjects/02-03-GPU-Components.md)
The building blocks of modern GPUs including texture mapping units (TMUs), render output units (ROPs), rasterization units, compute units, and command processors.

### [2.4 Console-Specific Architecture](../subjects/02-04-Console-Specific-Architecture.md)
Unique architectural features of Xbox, PlayStation, and Nintendo Switch GPUs, including unified vs dedicated memory architectures.
