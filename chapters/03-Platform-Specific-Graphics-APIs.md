# Chapter 3: Platform-Specific Graphics APIs

[← Back to Main Index](../README.md)

Each platform has its own graphics API with unique features and optimization opportunities. This chapter covers the major graphics APIs for PC and console development.

---

## Subjects in This Chapter

### [3.1 DirectX (PC/Xbox)](../subjects/03-01-DirectX.md)
Microsoft's graphics API for PC and Xbox, covering DirectX 11, DirectX 12, HLSL shader language, Xbox Series X/S optimizations, and Xbox One compatibility.

### [3.2 PlayStation](../subjects/03-02-PlayStation.md)
Sony's proprietary graphics APIs including GNM/GNMX for PS4/PS5, PSSL shader language, PlayStation 5 advanced features, and PS4 optimization strategies.

### [3.3 Nintendo Switch](../subjects/03-03-Nintendo-Switch.md)
Nintendo's NVN graphics API, handheld vs docked mode considerations, thermal management, and memory architecture specific to Switch.

### [3.4 Cross-Platform APIs](../subjects/03-04-Cross-Platform-APIs.md)
Modern cross-platform graphics APIs including Vulkan, Metal, and legacy OpenGL/OpenGL ES for broader platform support.
