# Chapter 4: Profiling and Analysis Tools

[← Back to Main Index](../README.md)

Profiling is essential for identifying performance issues. This chapter covers the comprehensive suite of tools available for graphics profiling on PC and consoles.

---

## Subjects in This Chapter

### [4.1 PC Profiling Tools](../subjects/04-01-PC-Profiling-Tools.md)
Essential PC profiling tools including PIX, RenderDoc, NVIDIA Nsight Graphics, AMD Radeon GPU Profiler, Intel GPA, and Unity's GPU profiler and frame debugger.

### [4.2 Console Profiling Tools](../subjects/04-02-Console-Profiling-Tools.md)
Platform-specific profiling tools including Xbox PIX, PlayStation Razor, PlayStation Tuner, Nintendo NNGFX Debugger, and LLGD (Low-Level Graphics Debugger).

### [4.3 Unity Built-in Tools](../subjects/04-03-Unity-Built-In-Tools.md)
Unity's native profiling suite including Profiler Window, Frame Debugger, Memory Profiler, Physics Debugger, and Audio Profiler.

### [4.4 Third-Party Profiling Tools](../subjects/04-04-Third-Party-Profiling-Tools.md)
Additional profiling solutions including Superluminal, Remotery, Tracy Profiler, and Optick for advanced performance analysis.
