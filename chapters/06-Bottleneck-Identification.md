# Chapter 6: Bottleneck Identification

[← Back to Main Index](../README.md)

Identifying bottlenecks is crucial for effective optimization. This chapter teaches you how to diagnose where performance is being lost.

---

## Subjects in This Chapter

### [6.1 CPU Bottlenecks](../subjects/06-01-CPU-Bottlenecks.md)
Identifying CPU-bound scenarios including main thread analysis, render thread analysis, script execution performance, physics simulation cost, and job system bottlenecks.

### [6.2 GPU Bottlenecks](../subjects/06-02-GPU-Bottlenecks.md)
Diagnosing GPU-bound scenarios including vertex-bound, fragment-bound, texture/memory-bound scenarios, API overhead, and command buffer overhead.

### [6.3 Memory Bottlenecks](../subjects/06-03-Memory-Bottlenecks.md)
Finding memory-related issues including VRAM usage and limits, texture streaming issues, asset loading stalls, garbage collection spikes, and memory fragmentation.

### [6.4 Bandwidth Bottlenecks](../subjects/06-04-Bandwidth-Bottlenecks.md)
Identifying bandwidth limitations including memory bus saturation, PCIe transfer bottlenecks, texture fetch latency, and render target bandwidth issues.
