# Chapter 8: Compression Techniques

[← Back to Main Index](../README.md)

Compression reduces memory footprint and loading times. This chapter covers platform-specific compression formats and techniques.

---

## Subjects in This Chapter

### [8.1 Texture Compression Formats](../subjects/08-01-Texture-Compression-Formats.md)
Platform-specific texture compression including DXT/BC (PC/Xbox), ASTC (Mobile/Switch), PVRTC, ETC2, platform-specific format selection, Crunch compression, normal map compression, and HDR texture compression.

### [8.2 Mesh Compression](../subjects/08-02-Mesh-Compression.md)
Mesh data compression techniques including quantization, delta encoding, Draco compression, Unity mesh compression, and vertex attribute packing.

### [8.3 Audio Compression](../subjects/08-03-Audio-Compression.md)
Audio compression formats including Vorbis, ADPCM, platform-specific codecs, and streaming audio compression strategies.

### [8.4 Data Compression](../subjects/08-04-Data-Compression.md)
General data compression including LZ4, LZMA, platform package compression, and AssetBundle compression options.
