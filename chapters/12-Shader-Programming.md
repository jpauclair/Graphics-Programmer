# Chapter 12: Shader Programming

[← Back to Main Index](../README.md)

Shader programming is the cornerstone of graphics development. This chapter provides complete coverage of shader development in Unity.

---

## Subjects in This Chapter

### [12.1 Unity Shader Languages](../subjects/12-01-Unity-Shader-Languages.md)
Available shader languages including HLSL, Cg (deprecated), ShaderLab, Shader Graph, and Visual Effect Graph shaders.

### [12.2 Shader Types](../subjects/12-02-Shader-Types.md)
Different shader types including surface shaders, vertex/fragment shaders, compute shaders, geometry shaders, tessellation shaders, and ray tracing shaders.

### [12.3 Lighting Models](../subjects/12-03-Lighting-Models.md)
Shader lighting including physically based rendering (PBR), custom lighting models, baked lighting integration, real-time GI integration, and stylized lighting.

### [12.4 Advanced Shader Techniques](../subjects/12-04-Advanced-Shader-Techniques.md)
Advanced shader features including alpha blending and testing, stencil buffer usage, custom post-processing, screen space effects, and shader includes/libraries.

### [12.5 Shader Compilation and Caching](../subjects/12-05-Shader-Compilation-And-Caching.md)
Shader compilation processes including runtime compilation, shader variant collection, shader cache warming, platform-specific compilation, and asynchronous compilation.
