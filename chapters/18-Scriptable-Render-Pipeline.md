# Chapter 18: Scriptable Render Pipeline (SRP)

[← Back to Main Index](../README.md)

Unity's Scriptable Render Pipeline provides complete control over rendering. This chapter covers SRP architecture and its implementations.

---

## Subjects in This Chapter

Core SRP concepts including render pipeline asset, render pipeline instance, scriptable render context, command buffers, and render graph.

### [18.2 Universal Render Pipeline (URP)](../subjects/18-02-Universal-Render-Pipeline.md)
Unity's URP including URP features and limitations, renderer features, custom render passes, URP optimization strategies, forward renderer, and 2D renderer.

### [18.3 High Definition Render Pipeline (HDRP)](../subjects/18-03-High-Definition-Render-Pipeline.md)
Unity's HDRP including HDRP features, material system, volumetric system, custom passes, and HDRP ray tracing.

### [18.4 Built-in Render Pipeline (Legacy)](../subjects/18-04-Built-In-Render-Pipeline.md)
Unity's legacy pipeline including built-in pipeline features, migration to SRP, and compatibility considerations.

### [18.5 Custom Render Pipelines](../subjects/18-05-Custom-Render-Pipelines.md)
Building custom pipelines including pipeline architecture, sorting and culling, custom lighting pipeline, and platform-specific paths.
