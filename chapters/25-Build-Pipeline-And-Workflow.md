# Chapter 25: Build Pipeline and Workflow

[← Back to Main Index](../README.md)

Efficient build pipelines and workflows are essential for team productivity. This chapter covers build optimization and asset pipeline management.

---

## Subjects in This Chapter

### [25.1 Build Optimization](../subjects/25-01-Build-Optimization.md)
Build optimization including build size reduction, code stripping, asset build pipeline, incremental builds, and player settings optimization.

### [25.2 Version Control](../subjects/25-02-Version-Control.md)
Version control for graphics assets including large file handling (Git LFS), binary asset management, shader cache management, build artifact storage, Unity Collaborate, and Perforce integration.

### [25.3 Continuous Integration](../subjects/25-03-Continuous-Integration.md)
CI/CD for game development including automated builds, automated testing, performance regression testing, multi-platform build systems, and Unity Cloud Build.

### [25.4 Asset Pipeline](../subjects/25-04-Asset-Pipeline.md)
Custom asset processing including asset import pipeline, custom asset importers, asset post-processors, and ScriptedImporter.
