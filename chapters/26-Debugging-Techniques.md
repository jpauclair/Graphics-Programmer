# Chapter 26: Debugging Techniques

[← Back to Main Index](../README.md)

Effective debugging is critical for graphics development. This chapter covers visual debugging, GPU debugging, and performance analysis.

---

## Subjects in This Chapter

### [26.1 Visual Debugging](../subjects/26-01-Visual-Debugging.md)
Visual debugging tools including debug shaders, wireframe rendering, overdraw visualization, LOD visualization, mipmap visualization, and UV layout debugging.

### [26.2 GPU Debugging](../subjects/26-02-GPU-Debugging.md)
GPU-level debugging including shader debugging, render target inspection, state inspection, GPU crash debugging, and shader compilation errors.

### [26.3 Performance Debugging](../subjects/26-03-Performance-Debugging.md)
Performance analysis including frame time analysis, GPU/CPU bubbles, memory leak detection, hitching and stutter analysis, and spike detection.

### [26.4 Unity Debug Tools](../subjects/26-04-Unity-Debug-Tools.md)
Unity's debugging features including Gizmos and Handles, Debug.DrawLine and DrawRay, OnDrawGizmos, and Scene View debugging.
