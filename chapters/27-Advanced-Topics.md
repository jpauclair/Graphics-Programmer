# Chapter 27: Advanced Topics

[← Back to Main Index](../README.md)

Cutting-edge graphics techniques push the boundaries of real-time rendering. This chapter covers ray tracing, VR, GPU-driven rendering, and next-gen features.

---

## Subjects in This Chapter

### [27.1 Ray Tracing](../subjects/27-01-Ray-Tracing.md)
Real-time ray tracing including DXR integration, ray tracing shaders, acceleration structures, hybrid rendering, ray tracing denoising, and performance considerations.

### [27.2 Virtual Reality](../subjects/27-02-Virtual-Reality.md)
VR rendering including VR rendering pipeline, single pass instanced rendering, multi-view rendering, VR performance optimization, foveated rendering, VR-specific anti-aliasing, and XR Plugin Framework.

### [27.3 GPU-Driven Rendering](../subjects/27-03-GPU-Driven-Rendering.md)
GPU-driven techniques including indirect drawing, GPU culling, mesh shaders, primitive shaders, GPU-side LOD selection, and visibility buffers.

### [27.4 Next-Gen Features](../subjects/27-04-Next-Gen-Features.md)
Emerging hardware features including mesh shading pipeline, variable rate shading, sampler feedback streaming, DirectStorage, work graphs, and hardware-accelerated GPU scheduling.

### [27.5 Nanite-Like Techniques](../subjects/27-05-Nanite-Like-Techniques.md)
Virtualized geometry including virtualized geometry systems, cluster rendering, software rasterization, and GPU-driven instancing.
