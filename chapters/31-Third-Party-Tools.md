# Chapter 31: Third-Party Tools and Libraries

[← Back to Main Index](../README.md)

Third-party tools can significantly accelerate development. This chapter covers popular tools and libraries for graphics programming in Unity.

---

## Subjects in This Chapter

### [31.1 Shader Tools](../subjects/31-01-Shader-Tools.md)
Visual shader editors including Amplify Shader Editor, Shader Forge, and Shader Graph extensions.

### [31.2 Rendering Extensions](../subjects/31-02-Rendering-Extensions.md)
Rendering plugins including post-processing stack extensions, HDRP/URP custom passes, and Rendering Plugin SDK.

### [31.3 Optimization Tools](../subjects/31-03-Optimization-Tools.md)
Asset optimization tools including Simplygon, InstaLOD, and texture packer tools.

### [31.4 Asset Creation Tools](../subjects/31-04-Asset-Creation-Tools.md)
Content creation integration including Substance Designer/Painter integration, Houdini Engine, SpeedTree, and Vegetation Studio.
